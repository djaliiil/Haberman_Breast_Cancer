1.	Initialement, veuillez installer les packages nécéssaires. (Python V==3.6)
		Pour les installer lancer la commande :
			"py -3.6 -m pip install -r requirements.txt"

2.	Maintenant on lance notre application :
		py -3.6 main.py
3.	Ensuite, ouvrez vous la page
		"127.0.0.1:5000"

Veuillez consulter le manuel de l'application !
